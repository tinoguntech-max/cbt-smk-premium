# 💬 Fitur Interaksi Guru - Siswa

## Overview

Menambahkan fitur interaksi real-time antara guru dan siswa:

### 1. Video Conference (Live Class)
- Video call dengan WebRTC
- Screen sharing untuk presentasi
- Chat selama video call
- Recording (opsional)
- Whiteboard virtual

### 2. Chat Real-time
- Chat per kelas
- Chat per materi
- Private chat guru-siswa
- Group chat untuk diskusi kelompok

### 3. Forum Diskusi
- Q&A per materi
- Q&A per ujian
- Voting untuk pertanyaan terbaik
- Guru bisa pin jawaban

### 4. Live Quiz
- Quiz interaktif saat live class
- Real-time leaderboard
- Instant feedback

## Teknologi

### Video Conference:
- **Jitsi Meet** (Open source, gratis, mudah integrate)
- **Agora.io** (Lebih advanced, ada free tier)
- **Daily.co** (Simple API, ada free tier)

### Chat Real-time:
- **Socket.io** (WebSocket untuk real-time)
- **Firebase Realtime Database** (Cloud-based)

### Recommended: Jitsi Meet + Socket.io
- Gratis & open source
- Self-hosted atau cloud
- Mudah integrate
- Support mobile
